<h1 align="center"> ToDoList 👋</h1>

> Application web de ToDoList en langage PHP, avec comme support bootstrap et base de donnée SQL.

### 🏠 [Homepage](https://github.com/Benjamin158/php-todo-app)


## Prérequis

- PHP 7.0~8.1
- Apache 2
- MySQL, PHPMyAdmin
- Editeur PHP
- Serveur Web
